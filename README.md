# Order Management Backend

## Tech Stack

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Swagger
- Maven

## Features

- User CRUD
- Product CRUD
- Order CRUD
- Place Order
- Get Orders By User
- Get Orders By Product
- Product Search
- Category Search
- Stock Management
- Validation
- Global Exception Handling
- Swagger Documentation

## API Documentation

http://localhost:8080/swagger-ui/index.html

## Database

MySQL

## Run

1. Clone repository
2. Configure MySQL
3. Run application
4. Open Swagger
